# Introduction to Deep Learning — Solving XOR with an MLP

**Author:** Nathan Feuerstein  
**Website:** [nathanfeuerstein.com](https://nathanfeuerstein.com)

---

## About

An interactive educational article on Deep Learning, covering the complete inner workings of a Multilayer Perceptron (MLP) built from scratch to solve the historical **XOR** problem — the challenge that almost killed artificial intelligence.

The article is fully bilingual **French / English**.

---

## Content

- Historical context of Deep Learning (1943–2017)
- How an artificial neuron works
- Layers, activation functions (Sigmoid), Loss Function (MSE)
- Backpropagation and learning rate
- Full MLP implementation in Python without any libraries
- 9 interactive quizzes (one per chapter)

## Interactive Features

- Live XOR demo with input sliders
- Real-time training visualization (gradient descent)
- Learning rate animation (optimal / too small / too high)
- Chapter navigation (desktop + mobile)
- FR / EN language toggle

---
